## 数据库设计文档

| 序号 | 数据表名      | 中文名称       |
| ---- | ------------- | -------------- |
| 1    | employee      | 员工信息       |
| 2    | category      | 菜品及套餐分类 |
| 3    | dish          | 菜品           |
| 4    | dish_flavor   | 菜品口味关系表 |
| 5    | setmeal       | 套餐           |
| 6    | setmeal_dish  | 套餐菜品关系   |
| 7    | user          | 用户信息       |
| 8    | address_book  | 地址簿         |
| 9    | shopping_cart | 购物车         |
| 10   | orders        | 订单表         |
| 11   | order_detail  | 订单明细表     |

### 1. employee

employee表为员工表，用于存储商家内部的员工信息。具体表结构如下：

| 字段名      | 数据类型    | 说明         | 备注        |
| ----------- | ----------- | ------------ | ----------- |
| id          | bigint      | 主键         | 自增        |
| name        | varchar(32) | 在职部门     |             |
| username    | varchar(32) | 用户名       | 唯一        |
| password    | varchar(64) | 密码         |             |
| phone       | varchar(11) | 手机号       |             |
| sex         | varchar(2)  | 性别         |             |
| id_number   | varchar(18) | 身份证号     |             |
| status      | bigint      | 账号状态     | 0:禁用，1:启用 |
| create_time | datetime    | 创建时间     |             |
| update_time | datetime    | 最后修改时间 |             |
| create_user | bigint      | 创建人id     |             |
| update_user | bigint      | 最后修改人id |             |

### 2. category

category表为分类表，用于存储商品的分类信息。具体表结构如下：

| 字段名      | 数据类型    | 说明         | 备注                 |
| ----------- | ----------- | ------------ | -------------------- |
| id          | bigint      | 主键         | 自增                 |
| name        | varchar(32) | 分类名称     | 唯一                 |
| type        | bigint      | 分类分组标识 | 如：1-第一组，2-第二组 |
| sort        | bigint      | 顺序         | 默认0                |
| status      | bigint      | 状态         | 0:禁用，1:启用       |
| create_time | datetime    | 创建时间     |                      |
| update_time | datetime    | 最后修改时间 |                      |
| create_user | bigint      | 创建人id     |                      |
| update_user | bigint      | 最后修改人id |                      |

### 3. dish

dish表为菜品表，用于存储菜品的信息。具体表结构如下：

| 字段名      | 数据类型      | 说明         | 备注        |
| ----------- | ------------- | ------------ | ----------- |
| id          | bigint        | 主键         | 自增        |
| name        | varchar(32)   | 菜品名称     | 唯一        |
| category_id | bigint        | 分类id       | 逻辑外键    |
| price       | decimal(10,2) | 菜品价格     |             |
| image       | varchar(255)  | 图片路径     |             |

| status      | bigint        | 售卖状态     | 0停售 1起售 |
| create_time | datetime      | 创建时间     |             |
| update_time | datetime      | 最后修改时间 |             |
| create_user | bigint        | 创建人id     |             |
| update_user | bigint        | 最后修改人id |             |

### 4. dish_flavor

### 8. address_book

address_book表为地址表，用于存储C端用户的收货地址信息。具体表结构如下：

| 字段名        | 数据类型     | 说明         | 备注           |
| ------------- | ------------ | ------------ | -------------- |
| id            | bigint       | 主键         | 自增           |
| user_id       | bigint       | 用户id       | 逻辑外键       |
| consignee     | varchar(50)  | 收货人       |                |
| sex           | varchar(2)   | 性别         |                |
| phone         | varchar(11)  | 手机号       |                |
| province_code | varchar(12)  | 省份编码     |                |

### 10. orders

orders表为订单表，用于存储C端用户的订单数据。具体表结构如下：

| 字段名                  | 数据类型      | 说明         | 备注                                                         |
| ----------------------- | ------------- | ------------ | ------------------------------------------------------------ |
| id                      | bigint        | 主键         | 自增                                                         |
| status                  | bigint        | 订单状态     | 1待支付：下单未付款2待商家接单：已付款，商家还没接单3商家接单,制作中：商家确认接单，正在做菜4待骑手取餐：商家出餐完成，骑手还没到店5骑手已取餐，配送中：骑手拿到餐，在路上，实时看定位6骑手已送达：骑手点送达，等待用户确认7订单已完成：系统自动确认收货8订单已取消：未接单退款、商家拒单、超时取消、售后全额退款 |
| user_id                 | bigint        | 下单用户     |                                                              |
| username                | varchar(32)   | 用户名称     |                                                              |
| consignee               | varchar(32)   | 收货人       |                                                              |
| phone                   | varchar(11)   | 手机号       |                                                              |
| address_book_id         | bigint        | 地址id       |                                                              |
| address                 | varchar(255)  | 地址         |                                                              |
| order_time              | datetime      | 下单时间     |                                                              |
| pay_method              | bigint        | 支付方式     | 1微信,2支付宝                                                |
| pay_status              | bigint        | 支付状态     | 0未支付 1已支付 2退款                                        |
| checkout_time           | datetime      | 结账时间     |                                                              |
| amount                  | decimal(10,2) | 实收金额     |                                                              |
| remark                  | varchar(100)  | 备注         |                                                              |
| cancel_reason           | varchar(255)  | 订单取消原因 |                                                              |
| rejection_reason        | varchar(255)  | 订单拒绝原因 |                                                              |
| cancel_time             | datetime      | 订单取消时间 |                                                              |
| delivery_status         | mediumtext    | 配送方式     | 1立即送出  0选择具体时间                                     |
| start_delivery_time     | datetime      | 配送时间     |                                                              |
| estimated_delivery_time | datetime      | 预计送达时间 |                                                              |
| delivery_time           | datetime      | 送达时间     |                                                              |

| 字段名          | 数据类型      | 说明     | 备注     |

| image           | varchar(255)  | 图片     |          |
| order_id        | bigint        | 订单id   |          |

| setmeal_id      | bigint        | 套餐id   |          |
| dish_flavor     | varchar(50)   | 口味     |          |
| number          | bigint        | 数量     | 默认1    |
| amount          | decimal(10,2) | 金额     |          |
| pack_amount     | bigint        | 打包费   |          |
